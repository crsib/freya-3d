Dependencies

  1. boost.filesystem 1.37.0 (it depends on boost.system 1.37.0)
  2. Lua 5.1.4 or LuaJIT 1.1.5 (note: to enable JIT engine usage define LUA_JIT_AVAILABLE)
  3. xerces-c++ 3.0
  4. OpenGL 2.1
  5. SDL 1.3 (svn version)
