Detailed Description

This class is an implementation of a task manager. It allows the application to use flexible, fast and reliable way to organize and execute various tasks in a multithreaded environment. This class provides control over the way of controlling number of threads, correctly execute tasks that must be executed exclusively on main thread and easing of implementation of multi-part or reentrant tasks
