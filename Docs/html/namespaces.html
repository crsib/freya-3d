Namespace List

Here is a list of all documented namespaces with brief descriptions:
coreThis namespace contains all classes related to engine core futures
core::drivermodelThis namespace contains all data related to Freya 3D engine drivermodel
core::filesystemThis namespace contains everything related to filesystem access
core::multithreadingThis namespace contains various abstractions, useful for multi-threaded programming and threads synchronization
core::taskmanagerThis namespace contains various classes connected with engines task management logic
rendererThis namespace contains all rendering capable functions
renderer::AlphaFunctionAlpha functions
renderer::ClientStateRepresent state of client application data pointers (mostly for OpenGL compatibility)
renderer::CubeTextureSideSides of cube textures (NOTE: to resolve cross-api portability the default coordinate system is treated in OpenGL way. E.g. the right triplet)
renderer::DataTypeType used for data storage
renderer::FogFunctionFog functions (for compatibility only)
renderer::FramebufferAttachmentFramebuffer attachments
renderer::FramebufferStatusStatus of framebuffer
renderer::FrontFaceThe bypass direction for front face
renderer::MatrixModeMatrix mode constants
renderer::PrimitiveThe type of primitives for rendering
renderer::RenderModeThe way the rendering is performed
renderer::ShaderTypeTypes of shader sources
renderer::StencilOpStencil test operations
renderer::TestFunctionConstants to various test functions
renderer::TextureClampingTexture clamping modes
renderer::TextureCoordTexture coordinates
renderer::TextureFilteringTexture filtering modes
renderer::TextureFormatExternal texture representation format
renderer::TextureGenModeTexture generation modes
renderer::TextureInternalFormatInternal texture formats (the way texture is represented inside accelerator)
renderer::TextureStorageTexture external storage format
renderer::TextureTypeTexture type constants
renderer::TextureUnitTexture unit constants
renderer::VBOAccessVBO access modes to mapped memory
renderer::VBOTypeVertex buffer object target (e.g. how the vertex buffer will be threated)
renderer::VBOUsageAlias for VBOType
resourcesThis namespace contains a various routines for maintaining resource management tasks
windowmanagerNamespace related to window managment and other window system specific futures of Freya 3D engine
