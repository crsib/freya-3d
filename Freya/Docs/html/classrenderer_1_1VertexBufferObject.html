Detailed Description

This is an abstraction of vertex buffer object (VBO). The VBO is a primitive, that is used for efficient memory exchange between RAM and VRAM an is often threaded as a source for vertex data. Another way of using VBO are different VRAM -> VRAM and RAM -> VRAM, such as implementation of render to vertex buffer technique and efficient loading of texture data (which could be implemented in renderer::Texture implemetation
